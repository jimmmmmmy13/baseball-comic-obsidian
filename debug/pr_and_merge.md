# PR、合併與網站更新（Day 2）

---

## 1) 我送出 PR 了，但網站沒有我的名字

最常見原因：PR 尚未被合併到 main。

網站只讀取 main，因此：
- PR 未合併 → 不會出現在網站底部

---

## 2) PR 合併了，但網站還沒更新

可能是瀏覽器快取：
- 請按 Ctrl+F5（Windows）或 Cmd+Shift+R（macOS）強制重新整理
- 或用無痕模式打開

---

## 3) PR 合併了，但還是沒有

請自查：
- 你的 `index.json` 是否放在正確路徑
- `index.json` 是否為有效 JSON（不是空檔、不是亂碼）
- 你的資料夾學號是否 9 碼英數

若仍不行，請截圖：
1) 你的資料夾結構
2) `index.json` 的檔案位置
3) PR 頁面
